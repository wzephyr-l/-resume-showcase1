## 💡 它是如何工作的？

```
用户上传 PDF
    ↓
验证文件（大小 < 5MB）
    ↓
提取 PDF 文本
    ↓
发送到 Claude API
    ↓

    ↓
生成美观的在线展示页面
    ↓
返回分享链接
    ↓
用户可以分享给雇主/朋友
```
