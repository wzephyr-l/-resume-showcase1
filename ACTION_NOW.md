### 步骤 2：推送代码到 GitHub（3 分钟）

**在 GitHub 创建新仓库：**
1. 访问 [https://github.com/new](https://github.com/new)
2. 仓库名称：`resume-showcase`
3. 描述：`AI-Powered Resume Parser`
4. Public（公开）
5. Create repository

**然后在 Windows 终端运行：**

```bash
cd D:\resume-showcase

git remote add origin https://github.com/YOUR_USERNAME/resume-showcase.git
git branch -M main
git push -u origin main
```

⚠️ **记住替换 `YOUR_USERNAME` 为你的 GitHub 用户名！**

---

### 步骤 3：在 Vercel 部署（5 分钟）

**访问 Vercel：**
1. 访问 [https://vercel.com](https://vercel.com)
2. 点击 **Sign Up** → **Continue with GitHub**
3. 授权 Vercel

**导入项目：**
1. 点击 **New Project**
2. 在列表中找到 `resume-showcase`
3. 点击 **Import**

**设置环境变量：**
1. 在部署前，找到 **Environment Variables**
2. 添加新变量：
   ```
   Key: ANTHROPIC_API_KEY
   Value: sk-ant-v1-your_actual_key_here
   ```
3. 点击 **Deploy**

## ❓ 遇到问题？

### 部署失败？

1. 检查 Vercel 的构建日志（红色错误信息）
2. 确保 `ANTHROPIC_API_KEY` 已设置
3. 尝试本地运行 `npm run build` 看看有没有错误

### 上传 PDF 时出错？

1. 确保 PDF 文件 < 5MB
2. 确保是有效的 PDF 文件
3. 检查 Claude API 配额是否足够

### 需要更多帮助？

- 查看 DEPLOYMENT_GUIDE.md
- 检查 Vercel 仪表板的日志
- 查看浏览器开发者工具 (F12) 的控制台
